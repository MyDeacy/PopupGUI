# PopupGUI
サーバーを開いてるPC上にGUIで案内を表示します  
***
![1](https://raw.githubusercontent.com/gigantessbeta/PopupGUI/master/image/screenshot.png)  
画像のようなことが可能になります。  
  
  
このコードとリソースををあなたのプラグインに貼り付けることで  
気軽にGUIを利用できます。  
  
  
  
  
現在, GUIに対応しているのはwindowsのみです。  
LinuxやMacにはコンソールメッセージという形で代用しています。  
できれば全部対応させたいです。
